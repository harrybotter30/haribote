# haribote

川合秀実著「[30日でできる！　OS自作入門](https://book.mynavi.jp/ec/products/detail/id=22078)」について学習するリポジトリ。

やりたいこと

- Linux 上で実施する
- 独自ツール群ではなく GNU toolchain を利用する
- GitHub の練習も行う

やらないこと

- オリジナルからの改変（環境を原因とする不可避の相違は除く）
- すでに知っていることの振り返り
